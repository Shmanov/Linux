2. Управление группами:
a) создать группы group1, group2 с использованием groupadd;
____________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudo groupadd group1
igor@ubuntu-virtual:~$ sudo groupadd group2
____________________________________________________________________________________________________
b) добавить user1 в группы group1, group2 разом при помощи одной команды usermod, не меняя его основной группы;
___________________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudo usermod -aG group1,group2 user1
group1:x:1003:user1
group2:x:1004:user1
________________________________________________________________________________________________

с) создать группу user1 с GID=1516 и сделать ее основной для пользователя user1
__________________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudo groupadd -g 1516 user1
igor@ubuntu-virtual:~$ sudo usermod -g user1 user1
igor@ubuntu-virtual:~$ sudo cat /etc/passwd
user1:x:1003:1516::/home/user1:/bin/bash
_______________________________________________________________________________________________

3. Создать пользователя с правами суперпользователя. 
___________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudo useradd -ou 0 -g 0 superuser
igor@ubuntu-virtual:~$ sudo passwd superuser
Новый пароль :
Повторите ввод нового пароля :
passwd: пароль успешно обновлён
____________________________________________________________________________________________
Сделать так, чтобы sudo не требовал пароль для выполнения команд.

___________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudoedit /etc/sudoers
[sudo] пароль для igor:
добавляем в файл sudoers строку:
igor ALL=(ALL) NOPASSWD: ALL
___________________________________________________________________________________________
4. * Используя дополнительные материалы, 
выдать одному из созданных пользователей право на выполнение ряда команд,
требующих прав суперпользователя (команды выбираем на своё усмотрение).
_____________________________________________________________________________________________
igor@ubuntu-virtual:~$ sudoedit /etc/sudoers

Cmnd_Alias      CMD = /sbin/groupadd, /sbin/usermod, /sbin/apt
user1 ALL=(ALL:ALL) CMD
